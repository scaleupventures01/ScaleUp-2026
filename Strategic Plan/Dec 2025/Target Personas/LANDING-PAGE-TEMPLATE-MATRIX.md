# Landing Page Template Matrix
## ScaleUp Ventures - Industry Landing Page Consolidation Analysis

**Analysis Date**: December 31, 2025
**Purpose**: Determine how many unique landing page templates are needed vs. how many can share a template with minor customizations

---

## Executive Summary

After analyzing all 40 target personas, I recommend **8 base templates** that can serve all industries with 80%+ shared content. Each template is designed around common pain points, value propositions, and buyer psychology rather than industry-specific features.

## Implementation Checklist

### For Each Template, Create:

- [ ] **Base HTML/CSS template** with placeholder sections
- [ ] **Content spreadsheet** with industry-specific swap fields:
  - Industry name
  - Market size stat
  - Key pain point stat
  - Association names (3-5)
  - Industry terminology (10-15 word swaps)
  - Hero image filename
  - Primary CTA text
  - Testimonial/case study placeholder
- [ ] **Image library** per industry (hero, icons, testimonials)
- [ ] **Landing page URL structure**: `/industries/[industry-slug]/`
- [ ] **Form fields** customized per industry (phone, email, company size)
- [ ] **Thank you page** per template group
