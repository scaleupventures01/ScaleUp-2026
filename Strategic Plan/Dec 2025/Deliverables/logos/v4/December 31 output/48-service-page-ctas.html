CTA Implementation Guidelines

Placement Best Practices

  • Hero Section: Primary CTA above the fold, secondary CTA below
  • Inline CTAs: Place after key value propositions or social proof
  • Sidebar CTAs: Keep visible during scroll, refresh content every 2-3 screens
  • Footer CTA: Strong closer with urgency or social proof
  • Exit-Intent: Trigger at 40% scroll + mouse exit, show once per session

Copy Guidelines

  • Action Words: Get, Calculate, Preview, See, Model, Start
  • Value Words: Free, Your, ROI, Savings, Growth, Certainty
  • Length: 3-6 words for primary CTAs, up to 8 for secondary
  • Subtext: Always include supporting copy below primary CTAs
  • Personalization: Use "My" and "Your" for ownership language

A/B Testing Priority

  1. Hero CTA copy - highest traffic, most impact
  2. Button color/style - primary vs animated
  3. Exit-intent offer - ROI preview vs strategy session
  4. Sidebar CTA position - top vs middle placement
  5. Mobile CTA type - fixed bottom vs floating

Tracking Requirements

  • Click Events: Track all CTA clicks with location context
  • Scroll Depth: Correlate CTA visibility with conversion
  • Device Type: Segment mobile vs desktop performance
  • Time on Page: Track engagement before CTA interaction
  • Exit-Intent: Track popup views, closes, and conversions
